# Kgaphola Telegram Reminder Bot

This is a Python-based Telegram bot that sends scheduled reminders to users.  
The bot helps users stay organised by delivering reminders for tasks, meetings, events, and daily activities directly on Telegram.

The project is designed as a real-world automation example using modern Python, async programming, and cloud deployment.

---

## Features

- Telegram bot built with python-telegram-bot (v22+)
- Sends scheduled reminders automatically
- Supports multiple users
- Runs continuously using long polling
- Secure handling of secrets using environment variables
- Cloud-hosted on Railway
- Logging of sent reminders

---

---

## Requirements

- Python 3.11 or higher
- Telegram Bot Token (from BotFather)
